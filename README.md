1. Create folder `assets/cv/`
2. Add your PDF as `assets/cv/DiogoGuillen_CV.pdf`
3. Update `cv.qmd` if you prefer a different name

## Publish (no terminal)
This repo includes a GitHub Action that builds and publishes the site automatically.
After you push changes, GitHub Pages updates in a couple of minutes.

## Local preview (optional)
